§ 01 Findings

Five things the desk isn't doing.

Read in order. The severities stack.

01 HierarchyHigh

Every action looks the same.

A duplicate company review and a user activation share the same row, the same chip, the same timestamp treatment. A board-visible item and a housekeeping item cannot feel equal. The table is a pure map() — no grouping, no sort by urgency, no visual weight.

Move
Group by type (Reviews, Activations, Housekeeping) with an eyebrow per group. Within a group, newest first.
02 CopyMedium

"Pending" is the wrong shape.

The counter chip reads 3 pending. Pending is the state of a request, not the posture the user is being asked to take. The desk should ask for your attention, not report the state of a queue.

Rewrite
Replace {n} pending with {n} on your desk. The count is the same; the posture reverses.
03 Empty stateMedium

"All clear" is a sigh, not a signal.

The empty state reads "All clear — no pending actions right now." That is two sentences to say nothing. The user wants to know what the rest of the system is doing, not that their inbox is empty. A quiet desk is an invitation to look around.

Rewrite
"Nothing for you today." followed by a single line inviting a stroll: "The register is quiet. The ledger is balanced. You could look in on the portfolio."
04 TimeLow

Relative time alone is not enough.

3m ago is precise but tells you nothing about when it was filed. For actions older than a day, the absolute date matters — audit logs, SLA clocks, and reviewers all need it. The relative time should live above, the absolute beneath.

Change
Show 3m ago as primary, · 14:02 Tue as a smaller line beneath, in monospace. Absolute time appears in tooltip today; pull it out.
05 Row affordanceLow

The row is a link pretending to be a row.

The whole row is clickable — onClick on the <tr> — but there's no visual chevron, no hover cue beyond the default table-row highlight. The action is hidden. A keyboard user with no mouse can't even tell the row is interactive until they tab into it.

Add
A right-aligned "Open →" micro-link on hover. Serves as the hover cue, the keyboard affordance, and the accessibility label in one move.
"

The home surface is not an inbox. It is a desk at the start of the day — papers stacked by someone who knew you'd come in.
